Install guide:
-------------

run:
* sudo apt-get install build-essential libssl-dev libboost-all-dev libgmp3-dev git libdb5.3++-dev libminiupnpc-dev libtool autotools-dev autoconf
enter:
* cd c3x/src/leveldb
run:
* chmod +x build_detect_platform
make:
* make -f makefile.unix
* strip c3xd
